# Screenmatch

Projeto em Java desenvolvido para **aprendizado** e prática de conceitos da linguagem.

## Funcionalidades

- Buscar série pelo nome  
- Listar episódios  
- Filtrar por ano  
- Buscar episódios por título  
- Calcular média de avaliações  

## Tecnologias

- Java  
- Spring Boot  
- HttpClient  
- Jackson
